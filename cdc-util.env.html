cdc-util.env

env->config

(env->config env config-keys)

Given a map of environment variables and collection of keys or tuples of [:env-key :config-key] returns a map of selected configuration values:

(env->config
 {:db-name "app-db"
  :db-host "db.example.com"
  :db-user "admin"
  :password "my-secret"
  :home "/users/example"
  ;; ... other env vars
  }
 [[:db-name :database]
  [:db-host :server]
  [:db-user :username]
  :password])
;=> {:database "app-db"
;;   :server   "db.example.com"
;;   :username "admin"
;;   :password "my-secret"}
